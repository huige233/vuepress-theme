huige的2023年年终总结

# 开篇

我们做得最好或最完美的事情,都以长时间反复练习得到充分掌握为前提,最终在不经意中像树上飘落的树叶一般降临到我们头上。 ——《梭罗日记》
1

光阴易逝，岁月如梭，转眼间2023年已经要结束了

要说的话，今年我是很不满意的，不但遇到了不少糟糕的事情，也把自己弄的一团糟的

不过也有值得庆幸的事情，比如死里逃生和拿到了一定的比赛名次

# 出行

今年趁着自己还有精力，跑了不少地方，也见识到不少的风景

# 3月10日 深圳

和两位好友在深圳碰了面，一起吃了一顿饭

上图拍摄于深圳机场的停车场，我们选择租车游玩

实话实说，玩的非常开心，和朋友一路上唠嗑打趣，说一些彼此之间知道的事情，开一些互相都能听懂的玩笑

# 4月28日晚 株洲

上图拍摄于广州火车站，19:13

因为一些原因，我当时出行选择的是无座坐到衡阳，后续再转高铁去目的地

20:55，列车出发，刚开始我还状态不错，后面开始感到疲惫和困倦，以及随身携带的充电宝电量不足

经过一番折腾之后，最终跑到了餐车去歇息，乘务人员非常热心，给了一些纸皮垫在地上，并且出借了充电头给我使用

直到4月29日的凌晨五点多，我才到达衡阳，稍作歇息，然后再次出发

之后则发生了一件非常有趣的事情，在我在长沙站等待的时候，去到了一家mcd中等待

它们的点餐机器直接崩溃了，在工作人员上班之后，也没有进行维修

最后选择的手机点餐，简单吃完之后，趴在桌子上睡了一会

最后历经折腾，也是到了目的地株洲，见到了几位朋友

晚上出去聚餐，寻找了一家自助餐厅，这家自助非常丰盛，而且价格也不贵

还玩了一下朋友的刀

5月1日，我们从株洲去到长沙，参加了长沙的漫展

在展子外面看到了一个 朱雀院椿 的痛车，上去和车主打了下招呼

在漫展里面，看到了很多的coser，也拍摄了不少的图片

本人临场犯社恐，一直怂恿朋友上去和coser交流，然后我在边上拍

但天下没有不散的宴席，最后还是要离开的

5月2日的晚上，我们在酒店边上的夜市买了许多的食物，聚在酒店里面一边聊天一边吃

# 7月20日 上海

这是一趟准备许久的旅行，早在六月就在计划了，直到七月初和朋友定好了行程

我的行程是从东莞南站出发，到达南昌西后转车到上海虹桥

上图是当时拍摄的东莞南站

沿途的风景让人心情愉悦，旅行的快乐就在其中

在南昌西急急忙忙的换乘之后，上了列车的我购买了一瓶饮料

怎么说呢，一次很亏的购物(

这瓶“橙汁”并不好喝，而且售价12元

时间兜兜转转，待到21:41,我终于到达了上海虹桥

一位朋友在我到达上海的第一天晚上，就拉着我去喝酒了

这里放出其中一张照片

7月21日，BW开场的第一天

临近九点，就和朋友一起赶去了会场

不曾想一路堵车，直到十点才开始进场

在折腾了将近半个小时之后，终于是进入了会场中

不愧是bw，有各式各样的coser，还有各种游戏的展摊

你甚至可以在bw打牌.jpg

还拍到了许多有趣的coser(比如你坤哥.jpg,不过受限于篇幅问题，就不放出来了)

到了16:40，原本正准备立场了，未曾想下起了暴雨

实属是我这辈子第一次见到这么大的雨，后面送我们的司机也说这是上海近年下的最大的雨

大概等了两个多小时雨才稍稍停歇，我们才得以离开会场

然后便是几次小小的聚会，我去找了在上海住的一位朋友，暂住在对方家里

然后按照自己的计划，去了上海的一些地方

上海外滩实话实说，我并不是很喜欢，因为人实在是太多了

另外一点就是，太多太多的高楼大厦，在下午并没有显得多少绚丽而是显得有些阴沉

在经历了一番折腾之后，我去到了东方明珠

在东方明珠的上层，可以看到整个上海的夜景

繁华的文明之星，甚至一度遮过了星空

7月27日 又被朋友拉去喝酒了(

倒是酒馆的名字和标牌很有意思

7月30日 是我在上海的最后一天

这一天去了cj展，然后就急匆匆的赶车去了

# 7月31日 青岛

当夜更深的时候
万物都在黑夜中失去了轮廓
唯有远方的孤灯常明
1
2
3

从上海虹桥出发，到济南西转车到青岛北

看着窗外的景色交替，我的心情也是随之变化

在济南转车的时候，有一位顺路的朋友，然后一起去吃了粉之后，一起前往车站等车

再经过一趟转车之后

终于，在凌晨四点到达了青岛

在青岛简单的玩了几天，见了朋友，打了maimai，也顺便把手机换成了现在的meizu20

受限于篇幅和个人原因，出行阶段就写到这里了

# 比赛相关

# OI

先说说OI部分吧，虽然已经退役很久了，但是不可否认的是，在这方面我还是有着一定的水平

值得开心的事情是，今年指导的朋友拿了一个csp一等和noip一等

也抓着朋友聊了一些自己喜欢的题目和对方喜欢的题目

不过普遍的，大家都不是很会写DP和背包类的题

# CTF

今年也算重归 CTF 的赛场了，虽然没有什么成绩，但是也算是有所收获

前后参加了 NewStarCTF 、 Geekgame 还有 Hackergame

NewStarCTF 偏向基础题，不过因为有人泄题的原因，当时中止了比赛，后续再关注的时候已经差不多结束了

Geekgame 的话，前后段的难度跨度非常巨大，前面就是新手都能简单解决的问题，但是后面的题难度一下子就非常大了

Hackergame 的话，我个人认为是今年打的最开心的一次

Hackergame 的题目难度是逐渐增加的，前面的一道题可能可以在三四分钟内解决，但是后面的题目几个小时都想不出来一个

汗流浃背了吧老弟，屯flag人太可怕了

经过了好几天的折腾，最后还是拿到了一个相对不错的名次

这是我的wp 欢迎大家来看看 link (opens new window)

# 生活

如果那天，我们双方的处境对调。我们之间的命运是否会有所不同。我会不会过着你的生活，而你会不会走我的老路。
1

# 人际关系

今年的人际关系只能说是一言难尽，有了再也不想看到的人，也有了不想失去的人

我不知道自己是不是真的变了，但是我知道，我不想再去和那些人打交道了

我清楚我的性格，我清楚我的脾气，我清楚我的底线，但是人际关系的原则问题，是不可以退让的

# 情感

今年也算是又有了一次刻骨铭心的感情经历

让自己受伤颇深，也让自己明白了，我终究是孤身一人

我宁愿相信没有来世。
我希望能在你身边更久。
1
2

# 健康

今年差不多整年都是在生病中度过的，年初的时候只是轻微的头晕和少量的咯血

等到了年中，随着几趟旅行的结束，我的身体出现了更严重的问题

我不知道自己是不是真的病了，但是我知道，如果不使用药物，我很难继续正常生活

等到后半年，发生了一些事情，导致我进了一次 ICU

至此之后，我的身体情况下行的非常严重，甚至连走路都会感到疲惫

目前在缓慢的休养中

# 2023年的感悟

很多时候，看的太远反而会让自己更加难受，看破不说破，是一种为人处世的艺术

很多代价并非简单的思考就可以明白自己是否能否接受的

如果那天，我们双方的处境对调。我们之间的命运是否会有所不同。我会不会过着你的生活，而你会不会走我的老路。
1

这句就是我的年度关键句，也是我今年最深的感悟

今年的生日过完，我也已然是一个20岁的年轻人了

也要更加习惯分别和关系的断裂

# 写在最后

虽然今年开玩笑说了很多次，或许活不过今年了

但是在真的到了12.31这天的时候，也还是有一种不真实感

写下这段话的时候，已经到了23:30了

很感谢这一年大家的陪伴和关心

祝各位2024年，平安喜乐，万事顺意！
